ObesityRisk_UI
ObesityRisk_UI is an interactive web application developed using Streamlit, designed to assess obesity levels based on user-provided personal characteristics and lifestyle habits. The application offers a simple and user-friendly interface, allowing users to input data and receive instant prediction results.

🔍 Main Features
User input of various attributes such as age, gender, height, weight, physical activity frequency, eating habits, etc.
Predicts obesity levels based on a pre-trained machine learning model.
Displays prediction results with detailed descriptions of each obesity level.
Integrates visual charts to analyze data and provide health improvement suggestions.
Simple, intuitive, and user-friendly interface.
📌 Technologies Used
Python: Main programming language.
Streamlit: Framework for building user interfaces.
Scikit-learn: Used for training and predicting machine learning models.
Pandas, NumPy: Data processing and analysis.
Matplotlib, Plotly: For creating interactive visualizations
